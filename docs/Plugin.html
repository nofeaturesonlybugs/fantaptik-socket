Plugin

Plugin is the interface for a Socket plugin.

A Plugin does not have to implement the entire interface. When an interface method is implemented the Socket instance will call the Plugin method at the appropriate time of its lifecycle and activity.

Plugins need to support a Socket property that will be assigned the instantiated Socket they are attached to.
